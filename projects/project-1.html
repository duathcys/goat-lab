프로젝트 개요

Raw 데이터 수집부터 데이터 마트 구성까지의 전체 ETL 파이프라인을 설계하고 구현했습니다. 데이터 정합성 검증 로직과 자동화 스케줄러를 도입해 수동 작업을 최소화하고, 실시간 대시보드를 위한 안정적인 데이터 적재 기반을 마련했습니다.

주요 작업

  • Raw 데이터 → 정제 → 데이터 마트까지 단계별 파이프라인 설계
  • 데이터 정합성 검증 및 오류 알림 로직 구현
  • 자동화 스케줄러 적용으로 수작업 제거
  • 실시간 대시보드 연동을 위한 적재 구조 최적화

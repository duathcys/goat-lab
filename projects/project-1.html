프로젝트 개요

자사몰과 네이버 스마트스토어 데이터를 활용해 운영 대시보드에 필요한 지표들을 자동 수집·적재·정제하는 데이터 파이프라인을 구축한 프로젝트입니다. GA4와 스마트스토어 API를 활용해 데이터를 수집하고, GCP 기반의 Cloud Function, BigQuery, Cloud Scheduler를 활용해 DataLake → DataWarehouse → DataMart로 이어지는 구조를 설계하고 자동화했습니다.

주요 작업

  • Raw 데이터 → 정제 → 데이터 마트까지 단계별 데이터 파이프라인 설계
  • 데이터 정합성 검증 및 오류 발생 시 Slack 알람 전송 로직 구현
  • Google Cloud Scheduler + Cloud Function을 활용한 자동화 스케줄링 구축 (오류 시 재시도 횟수 및 간격 설정으로 안정적 운영 보장)
  • Incremental Load 방식 전환으로 실시간·부분 업데이트 효율화
  • Raw Data 테이블 만료 설정 (2개월 자동 삭제)으로 저장 공간 효율화
  • 실시간 대시보드 연동 최적화를 위한 적재 구조 개선
  • 일정하지 않은 GA4 Raw Data 적재 시간 대응 및 안정적 파이프라인 운영

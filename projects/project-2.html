천안시 데이터 경진대회
Python Data Analysis 대상

프로젝트 개요

천안시 고령 운전자의 교통사고 심각도를 예측하기 위해 도로교통공단, 통신사, 신용카드, 네이버 지도 등 다양한 공공·민간 데이터를 자동으로 수집·전처리하고, 공간 정보 기반으로 통합 분석하였습니다. CatBoost, XGBoost, LightGBM 등 Gradient Boosting 계열 모델을 활용한 머신러닝 기반 사고 심각도 예측 모델을 설계·구축하고, 이를 바탕으로 정책 개선안을 제안하여 대상을 수상하였습니다.

주요 작업

  • 고령 운전자 교통사고 관련 공공·민간 데이터 수집 및 전처리
  • 천안시 주요 이용 시설(마트, 병원, 주유소, 은행) 크롤링 및 공간 정보 통합
  • 사고 유형·시간대·지역별 EDA 및 패턴 분석
  • 사고 심각도 예측을 위한 파생변수 생성
  • CatBoost, XGBoost, LightGBM 등 Gradient Boosting 계열 모델을 활용한 머신러닝 기반 사고 심각도 예측
  • 데이터 근거 기반 교통 정책 제안

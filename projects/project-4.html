프로젝트 개요

Google Marketing Platform(DV360) 플랫폼 데이터를 자동으로 수집·전처리하고 DB에 적재까지 하는 파이프라인을 구축하였습니다. 사용자의 데이터 요청은 Spring 기반 컨트롤러에서 처리하며, 플랫폼별 Cloud Run Job을 활용하여 크롤링을 병렬로 수행하도록 구현하였습니다. 이를 통해 데이터 정합성을 유지하고 안정적인 자동화 데이터 파이프라인을 마련하였습니다.

주요 작업

  • OTP 인증 자동화 및 세션 토큰 재사용 전략 구현
  • 세션 충돌 문제 해결 및 안정적인 크롤링 확보
  • 병렬 크롤링 적용으로 수집 속도 향상
  • 수집 데이터 DB 자동 적재 파이프라인 구축

Java

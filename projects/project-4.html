프로젝트 개요

로그인 세션 유지와 OTP 인증 자동화가 핵심 과제였습니다. Selenium 기반 크롤러에 세션 토큰 재사용 전략과 병렬 처리를 적용해 수집 속도를 대폭 향상시켰으며, 수집된 데이터를 DB에 자동 적재하는 파이프라인까지 완성했습니다.

주요 작업

  • OTP 인증 자동화 및 세션 토큰 재사용 전략 구현
  • 세션 충돌 문제 해결 및 안정적인 크롤링 확보
  • 병렬 크롤링 적용으로 수집 속도 향상
  • 수집 데이터 DB 자동 적재 파이프라인 구축

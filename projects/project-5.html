프로젝트 개요

SNMP 기반 네트워크 장비 데이터를 자동으로 수집하고 관리하기 위해 Python 기반 테스트 툴을 개발하였습니다. GET·GETSUB 통신을 구현하고, 다수 장비 데이터를 일괄 수집하도록 설계하였으며, 필수 입력값 누락 시에도 오류를 안정적으로 처리하도록 구현했습니다. 수집한 데이터는 엑셀로 자동 가공·저장되며, Windows·Linux 환경에서 실행 가능한 실행 파일로 배포하였습니다. 이를 통해 반복적인 수작업 없이 네트워크 장비 상태를 효율적으로 수집·보고할 수 있는 자동화 체계를 구축했습니다.

주요 작업

  • SNMP 요청·응답 수집 및 분석 로직 구현
  • 비개발자도 편리하게 실행할 수 있도록 윈도우·리눅스 환경별 실행 파일 빌드 및 배포
  • 결과 엑셀 자동 저장 기능 구현
  • 다수 장비 일괄 점검 자동화로 수작업 시간 단축

Python

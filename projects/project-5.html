SNMP Test Tool
Python Network Automation

프로젝트 개요

다수의 네트워크 장비에 SNMP 요청을 보내고 응답을 수집·분석하는 테스트 도구를 개발했습니다. GUI를 통해 비개발자도 쉽게 사용할 수 있도록 설계했으며, 결과를 엑셀 형식으로 자동 저장해 반복적인 수작업 시간을 크게 단축했습니다.

주요 작업

  • SNMP 요청·응답 수집 및 분석 로직 구현
  • 비개발자 친화적 GUI 설계
  • 결과 엑셀 자동 저장 기능 구현
  • 다수 장비 일괄 점검 자동화로 수작업 시간 단축

사용 기술

Python
pysnmp
tkinter (GUI)
openpyxl

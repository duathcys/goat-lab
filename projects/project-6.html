프로젝트 개요

Spring 기반 백엔드와 JavaScript 프론트엔드로 관리자 페이지를 풀스택 개발했습니다. 역할 기반 접근 제어(RBAC)로 메뉴·기능별 권한을 세밀하게 관리하고, 계층형 메뉴 구조를 동적으로 렌더링하는 로직을 구현해 운영 효율성과 보안성을 모두 높였습니다.

주요 작업

  • 사용자·권한·메뉴 관리 기능 풀스택 개발
  • RBAC 기반 메뉴·기능 단위 접근 제어 구현
  • 계층형 메뉴 구조 동적 렌더링 로직 설계
  • 보안성과 UX를 고려한 관리자 인터페이스 설계

사용 기술

Java / Spring
JavaScript
SQL / MyBatis
RBAC

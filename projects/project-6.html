프로젝트 개요

관리자의 효율적인 시스템 운영을 위해 사용자 관리, 권한 관리, 메뉴 관리 기능을 포함한 관리자 페이지를 개발했습니다. 특히 권한 기반 접근 제어와 계층형 메뉴 관리를 중심으로 설계하여, 사용자별 맞춤 기능 제공이 가능하도록 했으며, 실무 환경에서도 활용할 수 있는 안정적이고 확장 가능한 관리 시스템을 구축하는 것을 목표로 했습니다.

주요 작업

  • 사용자 관리 로직 구현 (계정 조회, 삭제, 권한 그룹 할당)
  • 메뉴 관리 기능 구현 (계층형 메뉴 생성/수정/삭제, 메뉴 순서 관리)
  • 권한 관리 기능 구현 (권한 그룹 생성/수정/삭제, 권한 부여, 접근 제어)
  • 권한 기반 UI 표시/비표시 및 권한 체크 기능 구현

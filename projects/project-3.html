프로젝트 개요

DBpia에서 제공하는 사회과학 분야 학술 논문 데이터를 활용하여 연구 트렌드를 정량적으로 분석하였습니다. 문맥 기반 키워드 분석으로 연구 주제 변화 흐름을 파악하고, 키워드 중요도 분석과 예측 모델을 결합하여 연구 트렌드의 동적 변화를 분석하였습니다. 이를 통해 과거와 현재 연구 흐름을 구조적으로 이해하고, 미래 연구 방향을 제시할 수 있는 데이터 기반 예측 체계를 구축하였습니다.

주요 작업

  • 학술 논문 메타데이터 수집 및 NLP 전처리
  • 키워드 빈도 변화 및 연구자 네트워크 분석
  • 시계열 모델 기반 키워드 중요도 예측
  • 인터랙티브 대시보드로 결과 시각화

수상

우수상 2025 SSU 데이터톤

사용 기술

Python
NLP (KeyBERT, SBERT)
ML(CatBoost, XGBoost, LightGBM)
Time-series Forecasting
